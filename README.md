## Build

### Build Webapp

-   `$ cd webapp`
-   > first time only `$ npm i`
-   `$ npm run build`

### Build Functions

-   `$ cd functions`
-   > first time only `$ npm i`
-   > > first time only and with **admin** privileges `$ npm run link`
-   `$ npm run build`

## Test

### Test Webapp

-   `$ cd webapp`
-   > first time only `$ npm i`
-   `$ npm run start`

### Test Functions

-   `$ cd functions`
-   > first time only `$ npm i`
-   > > first time only and with **admin** privileges `$ npm run link`
-   `$ npm run start`

## Deploy

Just `push` your `commit` with github and the `github workflow` _(inside `.github\workflows` folder)_ will **automagically** handle it.
